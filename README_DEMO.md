# README Demo Guide

## How to run the Next.js website

```bash
cd csk_steel_center_nextjs
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

`localhost:3000` means the demo is running on your own computer. For customer sharing, deploy to Vercel, Netlify, or your own server.

## Website pages

```text
/           Home / landing page
/services   Service details
/products   Product and material categories
/projects   Gallery layout for real project photos
/quote      Quote request demo form
/contact    Phone, Telegram, and map placeholder
```

## Demo notes

The quote form is a frontend demo. It creates a Telegram-ready message for the customer to copy.

For production, connect it to:

```text
Telegram bot
Google Sheets
Email
Backend API
Database
```
