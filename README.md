# CSK Steel Center Next.js Website

A Next.js + TypeScript demo website for CSK Steel Center.

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Pages

- `/` Home
- `/services` Services
- `/products` Product listing
- `/products/steel-sheets-plates` Product detail example
- `/products/steel-pipes-tubes` Product detail example
- `/products/bars-angles-frames` Product detail example
- `/products/gate-fence-parts` Product detail example
- `/products/cnc-decorative-panels` Product detail example
- `/products/custom-order-items` Product detail example
- `/projects` Project listing
- `/projects/cnc-decorative-panel-project` Project detail example
- `/projects/gate-fence-work` Project detail example
- `/projects/steel-material-supply` Project detail example
- `/projects/custom-fabrication` Project detail example
- `/quote` Quote form demo
- `/contact` Contact
